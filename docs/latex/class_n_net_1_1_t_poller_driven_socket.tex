\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class NNet\+::\+TPoller\+Driven\+Socket$<$ T $>$}
Socket type driven by the poller\textquotesingle{}s implementation. 

This class is used when socket operations (read, write, connect, etc.) are defined by the pollerâ€”in other words, when these operations depend on the underlying poller (e.\+g. for high-\/performance I/O using uring or IOCP).

Ordinary users should not worry about selecting the appropriate socket type. Instead, they should use the type defined by the poller, i.\+e. {\ttfamily typename} TPoller\+::\+TSocket. Depending on the poller, this may resolve to either \doxylink{class_n_net_1_1_t_socket}{TSocket} or \doxylink{class_n_net_1_1_t_poller_driven_socket}{TPoller\+Driven\+Socket}.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The poller type used by this socket. The poller must provide methods for asynchronous operations (such as Connect, Accept, Recv, and Send) and timer management. \\

\end{DoxyTemplParams}
