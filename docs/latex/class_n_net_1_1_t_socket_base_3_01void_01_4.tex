\doxysubsection{Detailed Description}
Internal base class encapsulating a socket (or file) descriptor and its poller. 

This class stores a file descriptor and a pointer to a poller (\doxylink{class_n_net_1_1_t_poller_base}{TPoller\+Base}) to support asynchronous I/O operations. It provides functionality to create a new socket descriptor (via \doxylink{class_n_net_1_1_t_socket_base_3_01void_01_4_a6ccea3dfc59c8a24c15a1c313e77784d}{Create()}) and to set it up (via \doxylink{class_n_net_1_1_t_socket_base_3_01void_01_4_a5123a941977e3426269c2d9cd9c70c13}{Setup()}). Additionally, it holds the descriptor and allows cleanup via \doxylink{class_n_net_1_1_t_socket_base_ac0deb2a1fd4b9b137eb0273a455e38e5}{Close()}.

\begin{DoxyNote}{Note}
This class is intended solely for internal use within the library. End-\/users should not interact directly with this class; instead, use the higher-\/level wrapper classes\+: \doxylink{class_n_net_1_1_t_socket}{TSocket}, \doxylink{class_n_net_1_1_t_file_handle}{TFile\+Handle}, \doxylink{class_n_net_1_1_t_poller_driven_socket}{TPoller\+Driven\+Socket}, or \doxylink{class_n_net_1_1_t_poller_driven_file_handle}{TPoller\+Driven\+File\+Handle}. 
\end{DoxyNote}
