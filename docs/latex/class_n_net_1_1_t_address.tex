\doxysubsection{Detailed Description}
A class representing an IPv4 or IPv6 address (with port). 

This class acts as a generic container for both IPv4 (sockaddr\+\_\+in) and IPv6 (sockaddr\+\_\+in6) addresses, providing a uniform interface for common operations (e.\+g., string conversion, retrieving raw data).
