\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class NNet\+::\+TPoller\+Driven\+File\+Handle$<$ T $>$}
Asynchronous file handle driven by the poller\textquotesingle{}s implementation. 

This class implements file I/O operations (e.\+g. read, write) using methods provided by the poller. In such scenarios, the poller controls the behavior of these operations (for example, when using io\+\_\+uring on Linux or IOCP on Windows).

Ordinary users should not choose a file handle type directly. Instead, they should use the type defined by the poller (i.\+e. {\ttfamily typename} TPoller\+::\+TFile\+Handle), which will resolve to either \doxylink{class_n_net_1_1_t_file_handle}{TFile\+Handle} or \doxylink{class_n_net_1_1_t_poller_driven_file_handle}{TPoller\+Driven\+File\+Handle} depending on the poller in use.
