\doxysubsection{Detailed Description}
Reads and stores DNS configuration from a file or an input stream. 

This class loads a list of nameservers from a DNS configuration file (by default, {\ttfamily /etc/resolv.conf}) or from any other input stream. The nameservers are stored as a vector of {\ttfamily \doxylink{class_n_net_1_1_t_address}{TAddress}} objects.
