\doxysubsection{Detailed Description}
Asynchronous file handle that owns its file descriptor. 

The passed file descriptor is owned by \doxylink{class_n_net_1_1_t_file_handle}{TFile\+Handle} and will be closed automatically when the object is destroyed.
